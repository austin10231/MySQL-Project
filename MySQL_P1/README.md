# Retail Sales Analysis SQL Project

## Overview
This project demonstrates basic SQL skills for retail sales data analysis.  
It involves creating a retail sales database, performing basic data cleaning, and using SQL queries to explore and analyze sales data.

## Database
Main Table: `retail`

The dataset contains transaction-level retail sales information including:
- Transaction ID
- Sale date and time
- Customer information
- Product category
- Quantity and price
- Total sales

## Key Tasks
1. **Database Setup** – Create the database and import the retail sales dataset.
2. **Data Cleaning** – Identify and remove records with missing values.
3. **Data Exploration** – Analyze sales data using SQL queries.
4. **Business Analysis** – Answer common business questions such as:
   - Total sales by category
   - Average customer age by product category
   - Top customers by total spending
   - Monthly sales trends
   - Best selling months per year

## Skills Demonstrated
- SQL querying
- Data filtering
- Aggregation (`SUM`, `AVG`, `COUNT`)
- Grouping (`GROUP BY`)
- Window functions
- Basic exploratory data analysis (EDA)

## Purpose
This project is designed as a beginner-friendly SQL project to practice data analysis using real-world style retail sales data.
